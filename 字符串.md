# 字符串

**字符串字典序排序**：

c++ 自带`sort()`函数

```cpp
//cpp
bool cmp(string a,string b){
    return a < b;
}

vector<string> strs(n);
sort(strs.begin(),strs.end(),cmp);
```

c语言 `strcmp()`函数

```c
void SortStrs(char ** strs,int n){
    char tmp[100] = {0};
    //对n个数据进行n-1次排序，每次冒泡出当前最大的
    for(int i = 1;i < n;i++){
        for(int j = 0;j < n - i;j++){
            if(strcmp(strs[j],strs[j+1]) > 0){
                strcpy(tmp,strs[j]);
                strcpy(strs[j],strs[j+1]);
                strcpy(strs[j+1],tmp);
            }
        }
    }
}
```
